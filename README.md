# Interakcija Čovek Računar 2025

Izvorni kod aplikacije sa vežbi iz predmeta Interakcija Čovek Računar na 4 godini Tehničkog Fakulteta Univerziteta Singidunum.

Aplikacija je razvijena koristeći sledeće tehnologije:
- Angular 20
- Bootstrap 5.3
- Fonts Awesome 7
- Axios HTTP Client
- Rasa Open Source

## Pokretanje projekta

Kako bi pokrenuli projekat potrebno je da posedujete Node 22 i Bun runtime na vašem računaru.
Aplikaciju pokrećete uz pomoć komande `bun start` ili `ng serve`
